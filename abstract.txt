Правила хорошего тона.
Типографика:
- не использовать слишком большие заголовки
- не использовать шрифт меньше 12 пикселей
- не белать слигком маленький/большой межстрочный интервал
- не растягивать буквы элементом transform
- не использовать более трез шрифтов на странице
- не смпользовать слишком маленький контраст (светлосерым по белому, темносерым по черному)
- использовать интервалы между символами осторожно
- не делать слишком маленьких отступов между абзацами дать воздуха дезайну
- не использовать капс без необходимости
- не использовать для основных блоков декоративный шрифт
- все ссылки, кроме ссылок меню, должны быть подчеркнуты и лучше использовать для посещенных ссылок - черный цвет
- если иерархия сайта содержит более трех уровней - не забыть о хлебных крошках
Графика, иконки, фотографии:
- не использовать стандартные фотографии, лучше сделать самому или найти более жизненные фотографии
- не использовать иконки, сделанные из фотографий
- все иконки - в едином стиле
- не увеличивать фото более ее оригинального размера
- не масштабировать графику не пропорционально
- не применять режим наложения слоев отличный от normal
- все иконки должны быть векторными, если растровые (фотошоп) - то большие
- все иконки в формате SVG
- цвет - не использовать чистые цвета, нужно добиться приятного оотенка
- не использовать более двух акцентных цветов на странице
- не использовать более двух темносерых/черных для текста
- акцентныи цветом закрышиваем только самые важные элементы на странице
Правила работы в графическом редакторе:
- называть слои со смыслом
- упорядочивать элементы в группы
- отдельно создать папку со шрифтами
- разрешение 72 пикселя/дюим
- использовать систему сеток (bootstrap)
- не создавать декоративных элементов, если в этом нет сысла
- придерживаться смысловой визуальной иерархии
- расстояние между внутренними элементами блока должнл быть меньше внешнего расстояние между блоками
- люди почти всегда заходят не на главную страницу, поэтому нужно хорошо подумать над шапкой и сайдбаром
- в футере можно разместить навигацию по сайту с потаенными местами
- продумать подвал, не делать подвал толлько с телефоном, сделать развернутыи
